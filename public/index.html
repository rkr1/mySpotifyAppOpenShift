    About mySpotifyApp

    This is an interview project. The requirements (intentionally paraphrased) include:

    • Build a web app that uses the Spotify API. The user should be able to:
      • Search for artists
        • Display a dropdown with search suggestions while the user types
      • Show a list of albums when the user clicks on a artist
      • Show the details of the album, including tracks and duration, when the user clicks on an album
    • Use ReactJS (not done)
    • Use another other framework, standard code, etc. This project uses:
      • Bootstrap 4
      • jQuery
      • Node.js
      • Autocomplete library from www.algolia.com
      • Example login code from Spotify
    • Preserves browser state, i.e. history buttons work (sort of works)
      • This sort of works, based on a made-up-hacked-up solution I came up with based on hashes.
      • The correct solution requires the use of router code.
    • Bonus
      • Compatible with all modern browsers
        • Intentionally used Bootstrap, which is designed for that purpose
      • Use of SASS/LESS and BEM would be appreciated (not done)
      • UX evaluated
        • I tried to make this app flow well without specifically copying any other examples that I saw on Spotify's developer site.
        • Better colors could be used.
        • Only retrieve the first x search matches. Didn't implement "get next".
